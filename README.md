# Campus-adv-ass
学园偶像大师 Aegisub字幕/CSV（by Hatsuboshi_Teleprompter）

## 使用之前
ASS文件用于视频字幕制作。  
CSV文件中包含AI翻译字幕/人工翻译以及校对，用于本地化插件。
